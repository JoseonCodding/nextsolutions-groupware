리엑트 사용법

PS C:\FrameG\apps\KDT_PJT> cd src
PS C:\FrameG\apps\KDT_PJT\src> cd main
PS C:\FrameG\apps\KDT_PJT\src\main> cd frontend
PS C:\FrameG\apps\KDT_PJT\src\main\frontend> npm start

cd src/main/frontend/
해당경로로 들어가 npm start 입력

npm install 를 실행하여
package.json에 명시된 의존성들이 모두 설치됩니다.

node_modules 폴더가 생성되면 설치성공

frontend 작업후 npm start를 해주셔야 3000번 포트 에서 정상확인가능합니다.
http://localhost:3000/
